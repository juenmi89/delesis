CSS Box Model이란?


CSS의 (박스모델)에 대해 알아봅시당~
border : 테두리
html Tag를 하나를 box로 취급해서 부피를 결정하는 것
block level element : 화면 전체를 쓰는 tag (h1과 같은) -> display를 통해서 자신의 부피만큼 사용하도록 할 수 있어 (inline)
inline element : 자신의 부피만큼만 쓰는 tag -> display를 통해서 자신의 부피만큼 사용하도록 할 수 있어 (block)
글자크기에 맞춰서 inline 처리!!!!!!!! -> display:inline-block;
